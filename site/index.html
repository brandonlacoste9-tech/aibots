How the bot works

Not a black-box “auto trader.” Every research turn is forced through tools, then balanced before you confirm.

price history → indicators → news → bull proposal → bear critic → you confirm
01

Forced research

History, indicators, and headlines run before any proposal. Numbers come from tools — not the model’s memory.

02

Bull + bear

One proposal, one skeptical critique on the same data. You review both sides in preflight.

03

Human gate

Proposals are record-only until you approve inside a TTL. The bot never submits live brokerage orders.

Get started

Open the web desk for freeform market Q&A, or run the API + CLI locally. 

git clone https://github.com/brandonlacoste9-tech/aibots
cd aibots
pip install -e ".[dev]"
cp .env.example .env   # XAI_API_KEY · optional FINNHUB_API_KEY

# Freeform market Q&A
python -m aibots ask "What is a PE ratio?"
python -m aibots ask "How does AAPL look on daily RSI?"

# Desk API (point the web UI at this URL)
python -m aibots serve --port 8080

# Forced research + bear critic
python -m aibots research AAPL
