Desk preview Example only
You What is a PE ratio?
Desk A P/E (price-to-earnings) ratio compares a stock’s price to its earnings per share. Roughly: how many dollars investors pay for each dollar of earnings. Higher often means growth expectations or a richer valuation; lower can mean cheaper or weaker outlook — always read it next to peers and the company’s story.

Get started

Start in the browser desk for freeform market Q&A. CLI and API are optional if you want to run things locally.

Open desk CLI on GitHub

Optional — run the API + CLI yourself:

git clone https://github.com/brandonlacoste9-tech/aibots
cd aibots
pip install -e ".[dev]"
cp .env.example .env   # XAI_API_KEY · optional FINNHUB_API_KEY

# Freeform market Q&A
python -m aibots ask "What is a PE ratio?"
python -m aibots ask "How does AAPL look on daily RSI?"

# Desk API (point the web UI at this URL)
python -m aibots serve --port 8080

# Forced research + bear critic
python -m aibots research AAPL
